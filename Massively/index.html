About me

Hello! I'm Nitya Nara, and I love everything about computer science. My interest in coding started in 9th grade when I first learned programming and realized how powerful it can be for solving problems and creating things that can make a real impact on society.
I've learned Python, HTML, and Java in my computer science classes, and I especially enjoy working with Java’s Object-Oriented Programming concepts. I have gained experience in these languages by building projects, from creating interactive games to designing apps that tackle real-world problems. My dream is to use technology to solve a real-world problem and eventually build a business around my skills.
Aside from coding, I spend time tutoring and encouraging middle and high school girls to explore computer science. I start by teaching them coding through Scratch, and then we gradually move on to using Python to build games.
